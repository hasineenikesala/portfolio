## 🌀 How the Rotating Border Animation Works
The dynamic glow ring surrounding the main profile image and tech stack graphics uses an absolute pseudo-element positioning layer:

1. **The Wrapper Container (`.logo-glow-wrap`):** Set to `position: relative` with a hidden overflow to maintain circular geometry (`border-radius: 50%`).
2. **The Glowing Pseudo-Element (`::before`):** Injected with a wide `linear-gradient(#00df9a, #38bdf8)` layer that extends beyond the card's dimensions.
3. **The Rotation Engine:** Driven by an active CSS Keyframe declaration:
   ```css
   @keyframes spinGlow {
       0% { transform: rotate(0deg); }
       100% { transform: rotate(360deg); }
   }
